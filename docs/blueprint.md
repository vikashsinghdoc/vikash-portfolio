# **App Name**: ProfilePilot

## Core Features:

- Profile Creation: Allow users to create and customize their profiles with personal information, work experience, education, and skills.
- Resume Builder: Provide a resume builder tool with templates and formatting options to generate professional resumes based on profile data. It should decide whether or not the resume needs some specific information.
- Portfolio Showcase: Enable users to showcase their projects and work samples in an interactive portfolio section.
- Contact Form: Include a contact form for visitors to easily reach out to the profile owner.
- Social Media Integration: Allow users to link their social media profiles (LinkedIn, GitHub, etc.) for increased visibility.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and professionalism.
- Background color: Light blue (#E8EAF6), subtly desaturated, provides a clean backdrop.
- Accent color: Violet (#7E57C2) to complement the primary blue and draw attention to key interactive elements.
- Headline font: 'Space Grotesk', a geometric sans-serif for titles; body font: 'Inter', a grotesque sans-serif for readability.
- Use clean, professional icons from a consistent set (e.g., Font Awesome) for navigation and key features.
- Design a clean, single-page layout with clear sections for easy navigation.
- Incorporate subtle transition animations for a polished user experience.
